# MinionsOS2 科学发现 Benchmark 设计

## 项目目标

设计一套针对 MinionsOS2 特点的自主科学发现 benchmark，结合 PaperBench 和 PaperWritingBench 的材料。

## 设计原则

1. **前沿性**: 基于主流、前沿的科研材料
2. **特色凸显**: 体现 MinionsOS2 在 Ethics 方面的特点
3. **开放性**: 设计开放式科研任务
4. **多元评估**: 从多个维度评判能力

## 场景选择

- 偏理论的研究
- 简单的机器学习任务
- 深度学习相关任务

## 团队讨论

本项目由 MinionsOS2 团队通过协作讨论完成设计方案。
